# zip-cli

>基于node和npm实现的命令行zip压缩工具。

## 安装

zip-cli 依赖 [Node.js](https://nodejs.org/en/) (>=9.x)：

```shell
$ npm install zip-cli -g
```

## 用法

```shell
$ zip [options]
```

示例：

```shell
$ zip -n filename   # 自定义文件名zip压缩

$ zip .             # 当前目录名zip压缩
```

## 基本命令

* 查看 zip 的帮助信息
  
```shell
$ zip -h

# or

$ zip
```

* 查看当前版本

```shell
$ zip -v
```
