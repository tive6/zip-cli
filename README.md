# TO-ZIP is a command-line zip compression tool based on node and NPM

## Installation

to-zip 依赖 [Node.js](https://nodejs.org/en/) (>=9.x)：

```shell
$ npm install to-zip -g
```

## Usage

```shell
$ zip [options]
```

eg：

```shell
$ zip -n filename   # 自定义文件名zip压缩

$ zip .             # 当前目录名zip压缩
```

## Basic command

* 查看 zip 的帮助信息
  
```shell
$ zip -h

# or

$ zip
```

* 查看当前版本

```shell
$ zip -v
```
